# InsightLens Documentation

This folder is the single source of truth for InsightLens user documentation. The in-app **Documentation** page is built directly from these markdown files at build time — if you update a file here, it shows up in the app on the next build.

There is no GitHub Pages site. Reading the docs means either opening InsightLens and clicking **Documentation** in the sidebar, or browsing the files below directly on GitHub.

## Contributing

If you spot something out of date or missing, open a pull request against the relevant file under `docs/`. Keep the writing plain, direct, and platform-specific where it needs to be. The in-app viewer is a lightweight markdown renderer — stick to headings, paragraphs, lists, links, inline code, and code blocks.
