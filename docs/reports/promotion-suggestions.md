# Promotion Suggestions

The **Promotion Suggestions** page surfaces units whose track record supports a case for academic promotion. It's designed for staff preparing evidence for a promotion application — their own or a colleague's.

## What it looks for

Promotion Suggestions combines several signals across multiple offerings of the same unit:

- Consistently high overall experience scores over several semesters.
- Healthy response rates so the numbers can be defended.
- Positive comment sentiment.
- Stable or improving trend, not a one-off spike.

The result is a ranked list of units — not a ranked list of individuals. InsightLens doesn't track instructors, so you'll need to map the units to the person whose case you're building.

## Using the list

Each entry shows the unit code, discipline, the number of qualifying offerings, a summary of the key metrics, and a link to the full [Unit Detail](../essential-workflow/unit-detail.md) page.

Click a row to drill into the evidence. The detail page shows the trend chart, individual offerings, and comments — everything you'd want to quote in a promotion document.

## Exports

The page supports both CSV and Markdown report exports, the same as [Performance Reports](performance-reports.md). The Markdown report is the more useful format for promotion packs: it includes a short narrative per unit and can be pasted into Word or a PDF template.

## Things to remember

- **This is evidence, not a decision.** The page identifies candidates whose data is strong; it doesn't know about teaching awards, research output, service load, or anything else in a promotion case.
- **Re-run after each import.** The list changes as new offerings land.
- **Check the comments.** High numbers with lukewarm comments is a weaker case than moderate numbers with strong qualitative feedback.
