# Unit Detail Page

The unit detail page is the deepest view InsightLens offers. Open it by clicking any unit from the Units list, the Dashboard, a Performance Report, or a Promotion Suggestion.

## Header

The top of the page shows the unit code, title, discipline, and the most recent offering. A back button returns you to wherever you came from.

## Trend chart

A line chart plots the unit's overall experience across every imported offering. Each point is a single semester. Hover for exact values.

Alongside the trend, headline stats show:

- Latest overall experience
- Latest response rate
- Total comments across all offerings

## Offerings list

Below the chart, every imported offering is listed chronologically. Each entry includes the year, semester, campus, overall experience, response count, and response rate. Clicking an offering expands the comment and question breakdown for that specific cohort.

## Comments with sentiment

Student comments are stored per-offering. Each comment is shown with an inline sentiment badge — positive, neutral, or negative — calculated locally with no external service call. Use the sentiment trend widget to see the balance over time.

## Course improvement

If a unit has multiple offerings, an action button lets you open the **Course Improvement** modal. This pulls together the trend, comment themes, and any AI-assisted suggestions into a short report you can export.

## Deleting an offering

If you need to remove a single imported survey — for example, because a corrected version was issued — use the delete control on the offering entry. The unit itself is preserved as long as it has at least one remaining offering.
